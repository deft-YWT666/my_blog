# 字符串模拟好题 [CF5A Chat Server's Outgoing Traffic - 洛谷](https://www.luogu.com.cn/problem/CF5A) 

## 题解

```c++
#include<bits/stdc++.h>

using namespace std;

int ans , num  ; //ans 为 总流量 , num 为 当前接受的总人数 
void solve()
{
	string s ;
	while(getline(cin , s)) //我们不知道这一题具体有几行操作,使用getline()函数读取整行可以省去许多不必要的麻烦,不过要记住cin是填在第一个参数的位置
	{
		if(s[0] == '+') num++ ; //直接按题目意思模拟 
		else if(s[0] == '-') num--;
		else
		{
			int pos = s.find(':') ;	
			/*
				如果忘了s.find() , 这部分写成这样也是可以的:
				int  pos = 0 ;
				while(s[pos] - ':') pos++ ;
			*/
			ans += (s.size() - 1 - pos) * num ;
		}
	}
	cout << ans << " " ;
}
int main()
{
	ios::sync_with_stdio(false) ; // 输入量可能很大 , 需要进行解绑操作 
	cin.tie(0) , cout.tie(0) ;
	solve() ;
	return 0;
} 
```
