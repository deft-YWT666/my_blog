# 李白打酒三种解法 [P8786 [蓝桥杯 2022 省 B\] 李白打酒加强版 - 洛谷](https://www.luogu.com.cn/problem/P8786) ]

## 1.二进制状态压缩枚举(50pts)

```cpp
#include<bits/stdc++.h>

#define int long long 

using namespace std ;

int n , m , ans ;

void solve ()
{
    cin >> n >> m ;
    int k = n + m - 1 ;
    for(int i = 0 ; i < (1LL << k)  ; i++)
    {
        int drink = 2 ;
        bool ok = true ;
        for(int j = k - 1 ; j >= 0 ; j--)
        {
          if(i >> j & 1)
          {
            drink *= 2 ;
            if(drink > m)
            {
                ok = false ;  break ;
            }
          }
          else 
          {
            drink --;
            if(drink < 0)
            {
                ok = false ; break ;
            }
          }
        }
        if(!ok) continue ;
        if(__builtin_popcount(i) == n && drink == 1) ans++ ;
    }
    cout << ans << endl ;
}
signed main()
{
    ios::sync_with_stdio(false) ;
    cin.tie(0) , cout.tie(0) ;
    solve() ;
    return 0 ;
}
```









## 2.dfs(50pts)

```cpp
#include<bits/stdc++.h>

#define int long long 
using namespace std ;

const int mod = 1e9 + 7 ;
int n , m ;
int dfs(int x , int y , int drink)
    //f(x , y , drink) = f(x - 1 , y , drink / 2) + f(x , y - 1 , drink + 1)
{
    if(x < 0 || y < 0 || drink < 0) return 0 ;
    if(x == 0 && y == 0 && drink == 2) return 1 ;
    int ans = 0 ;
    if(x > 0 && drink % 2 == 0)
    {
        ans += dfs(x - 1 , y , drink / 2) ;
    }
    if(y > 0 && drink <= m)
    {
        ans += dfs(x , y - 1 , drink + 1) ;
    }
    
    ans %= mod ;
    return ans ;
}
void solve()
{
    cin >> n >> m ;
    cout << dfs(n , m - 1 , 1) ;
}
signed main()
{
    ios::sync_with_stdio(false) ;
    cin.tie(0) , cout.tie(0) ;
    solve() ;
    return 0 ;
}
```
