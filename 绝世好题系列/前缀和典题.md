# 前缀和典题

##  1. [P3131 [USACO16JAN\] Subsequences Summing to Sevens S - 洛谷](https://www.luogu.com.cn/problem/P3131) 

### 题解

(1) 题意 : 意思就是让我们找出一个数组的区间 , 这种区间的和为 7 的倍数 , 这个区间的最大长度是多少?

(2) 思路 : 看到求和 , 区间和这种类型的题目 , 我们应该马上想到前缀和优化 : sum[i] = sum[i - 1] + a[i] . 

那我们不妨假设符合题意的区间[l , r] , 满足

​														 	(sum[r] - sum[l - 1]) % 7 == 0 .

 我们把这个式子进行化简: 

是不是就可以得到 sum[r] % 7 - sum[l - 1] % 7 == 0 , 然后再把减号移过去 , 得到 :

​															sum[r] % 7 == sum[l - 1] % 7

这个式子的意思是不是就是两个前缀和数组在 r 处和 l - 1 处模7的值是相等的 , 那我们是不是只要把整个sum数组模 7 , 然后找到相差最远的两个端点 , 就是本题的答案了 ! 这样做很简单, 因为模7的结果只可能是 0 ~ 6 , 因此我们只要用一个 哈希表 把第一次出现的余数的下标记录下来 , 然后我们每次遇到这个余数就把它的下标与其第一次出现的位置作差存到答案里 , 再用 max() 函数取一个最大 , 不难看出时间复杂度是 O(n) .

 

### 代码(c++)

```cpp
#include<bits/stdc++.h>

#define int long long 
//不开 long long 见祖宗!!!
using namespace std ;

const int N = 5e4 + 10 ; 

int  n , a[N] , ans , sum[N] ;
int mp[7] = {0 , -1 , -1 , -1 , -1 , -1 , -1} ;
//余数为 0 第一次出现的位置是 0 , 因为如果sum[i]本身就能被 7 整除的话, 最大长度一定是 i
void solve()
{
	cin >> n ;
    
	for(int i = 1 ; i <= n ; i++)
	{
		cin >> a[i] ;
		sum[i] = (sum[i - 1] + a[i]) % 7 ;
	}
    //经典构造前缀和 , 再把sum数组模 7
    
	for(int i = 1 ; i <= n  ; i ++)
	{
		if(mp[sum[i]] == -1) mp[sum[i]] = i ;
        //记录第一次余数出现的位置
		else 
		{
			ans = max(ans , i - mp[sum[i]]) ;
		}
        //如果不是第一次出现 , 或者本身从0 ~ i就是 7 的倍数 , 就直接更新答案就行了 
	}
	cout << ans << " " ;
 } 
signed main()
{
	ios::sync_with_stdio(false) ;
	cin.tie(0) , cout.tie(0);
	solve() ;
	return 0 ;
}
```



##  P8649 [蓝桥杯 2017 省 B\] k 倍区间 - 洛谷](https://www.luogu.com.cn/problem/P8649) 

### 题解

本题和上一题换汤不换药 , 问区间和为 7 的倍数的最大值 , 改为了问区间和为 k 倍的数量 .  不难看出 , 新出现第 n  个余数 , 答案就要加 n - 1(余数为 0 , 比较特殊 , 答案要加 n ) , 

举个例子 当 k = 3 , n = 6 时 sum数组模k 为 0 1 2 0 1 2 (下标 1 ~ n )

我们遍历到 下标 1 时 , 第 1 次出现余数 0 , 答案 + 1 ,  遍历到下标 4 时 , 第二次出现余数 0 , 答案 + 2 ,遍历到下标 2时 , 第一次出现余数1 答案 + 0 , 遍历到下标 5 时 第二次出现余数 1 , 答案 + 1 , 以此类推.

### 代码(c++)

```cpp
#include<bits/stdc++.h>

#define int long long 

using namespace std ;

const int  N = 1e5 + 10 ;
int n , a[N] , sum[N]  , ans  , k , mp[N]; 

void solve()
{
	cin >> n >> k;
	for(int i = 1 ; i <= n ; i++)
	{
		cin >> a[i]  ;
		sum[i] = (sum[i - 1] + a[i]) % k ;
	}
	mp[0] = 1 ;
	for(int i = 1 ; i <= n ; i++)
	{
		ans += mp[sum[i]] ;
		mp[sum[i]]++ ;
	}
	cout << ans << ' ' ;
}

signed main()
{
	ios::sync_with_stdio(false) ;
	cin.tie(0) , cout.tie(0) ;
	solve() ;
	return 0 ;
}
```
